## 자료구조 : 요리재료 + 다듬는 방법

- 선형 자료구조
  - 리스트
    - 선형리스트(= 순찰리스트) : 배열, 빈틈이 없다.
        장점: 공간 절약된다.(= 비용 절감), 전체 접근 빠름
        단점: 오버헤드가 생긴다.
    - 단순 연결 리스트: Node(= Data + Link)
        장점: 삽입/삭제 시 오버헤드가 발생하지 않는다.
        단점: 공간이 더 필요(= 비용이 비쌈), 전체 접근 느림
        마지막 노드? Link == none
    - 원형 연결 리스트: 마지막 노드 link == head

  - 스택

  - 큐
    - 순차 큐(= 일반 큐)
    - 원형 큐(= 환형 큐)

- 비선형 자료구조
